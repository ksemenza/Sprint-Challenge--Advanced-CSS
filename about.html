Question 1

What is the difference between an adaptive website and a fully responsive website?
Responsive reacts to any changes in "response" to the dimensions of the current browser. Adaptive make changes according to specific dimensions helping the view to 'adapt' to a different devices. dimensions

Question 2

Describe what it means to be mobile first vs desktop first.
It means that the original formatting is based off spec of a mobile device and is adapted to fit a website's dimensions

Question 3

What does font-size: 62.5% in the html tag do for us when using rem units?

Question 4

How would you describe preprocessing to someone new to CSS?
It is a compiler that integrates an imported file and converts it into a CSS format. It is used to create a more consistent layout and gives the developer more efficient control over any future changes.
